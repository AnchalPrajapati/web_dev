Frontend Development

CSS Javascript

Frontend development is the development of the graphical user interface of a website, through the use ofHTML, CSSand Javascript, so that users can view and interact with the website.

  • HTML: The Hyper Text Markup Language or HTML is the standard markup language for document to be displayed in a web browser.
  • CSS: Cascading Style Sheet(CSS) is a style sheet language used for describing the presentation of a document written in amarkup language such as HTML or XML.

  1. Pure CSS

  2. Bootstrap(Framework)

  3. Tailwind(Framework)

  4. Sass(Preprocessor)

  • Javascript: Javascript often abbreviated JS, is a programming language that is one of the technologies of the World Wide Web, alongside HTML and CSS. It use to add functionality in the website.

  1. Vanilla JS

  2. jQuery (Library)

  3. React (Framework)

  4. Anular (Framework)

  5. Vue(Framework)

  6. Typescript (Preprocessor)
